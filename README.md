>
> - **한 주에 4문제 풀기**

>   - 개인 풀이 2문제 : 각각 실버 3, 2이상 (solved.ac 상위 100 이내)
>   - 토요일 자정까지 push, PR
> - **코드리뷰** 
>   - 4문제 PR에 코멘트 남기기
> - **일요일 21:00 온라인 스터디**
